1. Systemvoraussetzungen

	
	1.1 Benötigt

		- PHP 4.0 oder höher
		- 1x MySQL Datenbank
		

		1.1.1 php.ini

			- magic_quotes_gpc = OFF
			- magic_quotes_runtime = OFF
			- magic_quotes_sybase = OFF

		// Wenn die php.ini nicht bearbeitet werden kann, erstelle eine .htaccess und füge "php_flag magic_quotes_gpc Off" hinzu!


	1.2 Empfehlenswert

		- SAFE_MODE = OFF
	

2. Installation

	
	2.1 Erstelle eine Datenbank und imprtiere die SQL-Datei - ts3admin.sql -
	2.2 Lade alle Daten im ts3admin-Order auf den Webserver
	2.3 Editiere die Datenbank-Einstellungen in der Datei /backend/inc/constant.php
	2.4 Setze die Rechte der folgenden Order auf 0777
		
		- /backend/temp
		- /backend/temp/1
		- /uploads/customer
		- /uploads/backup

	2.5 Füge die Webserver IP der query_ip_whitelist.txt hinzu
	2.6 Nun kannst du dich mit dem Benutzer "admin" und dem Passwort "avatar3d" einloggen.


3. Erste Schritte

	
	3.1 Gehe in den Adminbereich
	3.2 Füge einen Rootserver hinzu
	3.3 Füge nun eine Instanz unter dem Menüpunkt "Voiceserver" hinzu
	3.4 Weise nun unter dem Menüpunkt "Voices. zuweisen" bestehende Server einem Benutzer zu
	3.5 oder erstelle neue Server
     !! 3.6 Ein virtueller Server kann erst über den Adminbereich administriert werden, sobald der Server einem Benutzer zugewiesen ist!!!
